# Final Style Direction: Modern Operational Command Center

## Primary Brand Feel

Cinematic enterprise operations platform.

Real references:
- ClickUp
- Airtable
- Asana Enterprise
- Linear

## Typography

### Headings

Font: Clash Display

Use only for:
- Homepage hero
- Project page hero titles
- Featured project headings
- Major section titles

Weight:
- 600-700 only

Tracking:
- Slightly tight
- Not ultra-condensed

### Body / UI / Dashboard Text

Font: Inter

Use for:
- All body text
- Dashboard labels
- Metadata
- Project descriptions
- Tables
- Pills
- Nav
- Buttons

Weight system:
- 400 = body
- 500 = labels
- 600 = emphasis
- Avoid 700 except headings

## Section Styling

### Hero Sections

- Minimal borders
- Massive typography
- Cinematic spacing
- Asymmetrical composition
- Strongest contrast

### Content Sections

- Cleaner
- More editorial
- Fewer visible boxes
- Stronger typography hierarchy

### Dashboard / Demo Sections

- Darker than surrounding page
- Controlled visual depth
- Subtle elevation
- Strongest visual focus

## Button Styles

### Primary CTA

Style:
- Indigo fill
- Light text
- Medium radius
- No gradients

Hover:
- Slightly brighter indigo
- Tiny elevation increase

### Secondary CTA

Style:
- Ghost button
- Thin border
- Muted text

Hover:
- Subtle surface fill only

## Border Radius

- Large containers: `16px`
- Cards: `12px`
- Pills: `999px`

## Shadow System

Do not use:
- Giant floating shadows
- Glow halos
- Glass blur

Use:
- Soft operational depth: `0 8px 24px rgba(0,0,0,0.18)`
- Light mode depth: `0 6px 18px rgba(15,23,42,0.06)`

## Homepage Style

Feel:
- Executive operational platform intro

Layout:
- Split hero
- Left narrative
- Right operational visual
- Strong asymmetry
- Featured system below

## Work Page Style

Feel:
- Enterprise systems showcase

Layout:
- Featured flagship project
- Secondary projects below
- Left-aligned project info
- Cinematic thumbnails
- Operational UI previews

## Project Page Style

Feel:
- Enterprise case study

Layout:
- Hero
- Proof strip
- Live demo
- Architecture narrative
- Workflow logic
- Technical depth

## Live Demo Style

Critical:

Not:
- Abstract graphics
- Full ERP screenshots

Yes:
- Curated operational product shots

Meaning:
- Believable enterprise UI
- Reduced density
- Strong focal hierarchy
- Cinematic framing
- Premium operational composition

## Thumbnail Style

Feel:
- Executive operational preview

Rules:
- 70% hero visual
- 30% support signals
- Realistic UI excerpts
- No fake infographic look
- No compressed dashboard chaos

## Spacing System

Massive rule:
- Fewer boxes
- More breathing room

Premium feel should come from:
- Hierarchy
- Rhythm
- Spacing
- Composition

Not:
- Effects
- Gradients
- Glow

## Animation Rules

Allowed:
- Subtle hover elevation
- Soft opacity transitions
- Slight movement

Not allowed:
- Neon glow
- Floating motion
- Glass blur
- Parallax overload
- Animated gradients

## Overall Target Feel

The final site should feel like a premium enterprise operational intelligence platform company where Ashton Medina is the architect behind the systems.
