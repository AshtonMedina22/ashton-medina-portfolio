# Ashton Medina Portfolio

Portfolio website showcasing work as a Full-Stack Platform Engineer.

## About

This portfolio showcases three major platform engineering projects:
- **Sales-to-Delivery Automation Platform** - Automated cross-module workflow system
- **Vendor Lifecycle & Compliance Platform** - Secure vendor management workflow system
- **Operational Intelligence Platform** - Computed KPI and aggregation engine

## Tech Stack

- **Framework**: Next.js
- **Design System**: Once UI
- **Content**: MDX-based content system
- **Deployment**: Vercel-ready

## Getting Started

**1. Install dependencies**
```bash
npm install
```

**2. Run dev server**
```bash
npm run dev
```

**3. Edit content**
- Personal info: `src/resources/content.tsx`
- Projects: `src/app/work/projects/*.mdx`
- Blog posts: `src/app/blog/posts/*.mdx`

## Features

- Clean, professional design
- MDX-based content system
- SEO optimized with automatic metadata
- Responsive layout
- Project case studies with detailed technical content
